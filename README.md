# AMR_SEPSIS
🧬 Genomic Characterization of Sepsis Pathogens (Klebsiella pneumoniae &amp; Escherichia coli)  This repository contains bioinformatics pipelines and analysis scripts used for the genomic characterization of sepsis-associated bacterial pathogens, with a focus on Klebsiella pneumoniae and Escherichia coli isolated from bloodstream infections.
